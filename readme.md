# King of bots
基于SpringBoot的AI对战平台
## 项目结构
![1cf35c2a75f2cab2f9e64d797e9ddcbf](./assets/1cf35c2a75f2cab2f9e64d797e9ddcbf.jpg)
### 实现模块
* 用户中心
  * 注册
  * 登录
  * 我的Bot
  * Bot的记录
* PK
  * 匹配界面（微服务）
  * 实况（websocket）
  * 真人PK（websocket）
  * Bot代码执行（微服务）
* 对战列表
  * 录像回放
* 排行榜
